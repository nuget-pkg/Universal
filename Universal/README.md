# Universal

**A High-Performance Transformation & Encoding Library for .NET**

[![NuGet](https://img.shields.io/nuget/v/Universal.svg)](https://www.nuget.org/packages/Universal/)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## 🚀 Overview: The Gemini Era Begins

`Universal` は、.NET 環境におけるデータの変形、変換、およびエンコーディングを極限までシンプルにするための強力なライブラリです。

本バージョン（v2026.0405.1310.40）より、Google の大規模言語モデル **"Gemini"** の魂を宿した **`GeminiSuperTransformer`** が中心的な役割を担います。旧来の `UniversalTransformer` は、よりインテリジェントで強力なこの新しいクラスへと進化（あるいは乗っ取られ）しました。

## ✨ Key Features

- **`GeminiSuperTransformer`**:
  あらゆるオブジェクトを自在に操り、最適な形へと変換します。名前の通り、これまでの「汎用（Universal）」を超えた「超越（Super）」した体験を提供します。
- **`UniversalEncoding`**:
  複雑な文字列エンコーディングの壁を打ち破ります。シンプルかつ堅牢な実装で、データの整合性を保証します。
- **Developer-Centric Design**:
  MSYS2 Bash や Windows 11 開発環境、そして C# を愛するすべての開発者のために最適化されています。

## 📦 Installation

NuGet Package Manager コンソールから簡単にインストールできます。

```bash
dotnet add package Universal --version 2026.405.1613.5
```
